MySQL Part 1

  • Database is an organized structure used to store and access information securely on a remote system.
  • We use MySQL to implement it into a system/website because of its efficiency in time required to execute commands and ability to store/access data.
  • Some advantages of using a relational database system resides in the concepts of primary and foreign keys.
  • Many tables can be created within one database that are related through the primary and foreign keys.
  • The three different types of relationships are "one to one," "one to many," and "many to many."
  • A key is any attribute that a table may have. A foreign key acts as a link between tables in a database and there can be any number of them.
  • A primary key must be a unique column and only one is allowed in a table.
